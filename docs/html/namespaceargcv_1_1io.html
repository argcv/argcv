◆ Dirname()

string argcv::io::Dirname ( string_view  in)
inlinenoexcept

Dirname returns all but the last element of path, typically the path's directory.

After dropping the final element using Split, the path is Cleaned and trailing slashes are removed. If the path is empty, Dirname returns ".". If the path consists entirely of slashes followed by non-slash bytes, Dirname returns a single slash. In any other case, the returned path does not end in a slash.
